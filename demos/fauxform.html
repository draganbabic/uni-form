Form Title

The .form-title class is reserved for styling the title of the form. Any of the heading elements are recommended for this instead of the legend element.

Alternate Form Fields Layout

Triggered by class .inline-labels, the alternate form fields layout is used for laying out forms where you want the label to be in line with the form field.

This is a form hint.

.small on .input-text

.medium on .input-text

Columns

Base class is .uni-form-row and classes that determine the number of columns are .uni-form-columns-2 and .uni-form-columns-3. Note the usage of width class .large on the input in order to make it span the full width of the column and force the .form-hint to take full width as well.
